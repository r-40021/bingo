自動保存機能を搭載したビンゴマシーン

(一部のブラウザでは自動保存機能が利用できません。)

# 特長
- データが自動で保存されます。 (一部のブラウザを除く)
- 直感的に使えます。
- 抽選を1つ前からやり直す機能を搭載しています。
- ライトモード/ダークモードを切り替えられます。
- 読み込みが高速です。
