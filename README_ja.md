自動保存機能を搭載したビンゴマシーン

(一部のブラウザでは自動保存機能が利用できません。)

# 特長
- データが自動で保存されます。 (一部のブラウザを除く)
- 直感的に使えます。
- 「戻る」ボタンと「進む」ボタンを搭載。抽選を特定のタイミングからやり直すことができます。
- ライトモード/ダークモードを切り替えられます。
- ボタンを大きめにしたため、スマートフォンでの操作も快適です。
